<p>Write a query to find the node type of <em>Binary Tree</em> ordered by the value of the node. Output one of the following for each node:</p>

<ul>
    <li><em>Root</em>: If node is root node.</li>
    <li><em>Leaf</em>: If node is leaf node.</li>
    <li><em>Inner</em>: If node is neither root nor leaf node.</li>
</ul>
